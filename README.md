# 🚢 Titanic Survival Prediction (Machine Learning Project)

This project predicts whether a passenger survived the Titanic disaster using Machine Learning (Logistic Regression).

---

## 📊 Dataset
- Titanic dataset from Kaggle / CSV file
https://www.kaggle.com/datasets/yasserh/titanic-dataset
- Features include: Age, Sex, Pclass, Fare, Embarked, etc.

---

## ⚙️ Steps Performed

1. Data Loading
2. Missing Value Handling
3. Feature Engineering
4. Encoding Categorical Variables
5. Train-Test Split
6. Model Training (Logistic Regression)
7. Prediction
8. Model Evaluation

---

## 🤖 Model Used
- Logistic Regression

---

## 📈 Performance
- Accuracy: ~81% (varies slightly)

---

## 🛠️ Libraries Used
- pandas
- scikit-learn
